# ToDo Mindmap — App Standalone PWA

## Description
App de productivité combinant todolist + mindmap visuelle. PWA installable sur téléphone. Zero auth, zero backend, tout en localStorage.

## Stack
- React 19 + Vite
- localStorage (pas de Supabase, pas de backend)
- PWA (vite-plugin-pwa)
- Déployé sur Railway via Docker + nginx

## Origine
Fork du module mindmap de fxscale-dashboard, transformé en app standalone.
